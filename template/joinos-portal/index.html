Overview

The Joinos for Parents portal is built on Bootstrap 3

Typography

How text is displayed

Text Size

Text sizes are controlled by the html structure of the page h1 to h5.

Text Alignment

Text can be aligned using the following 3 classes.

  • .text-left will align the text to the left.
  • .text-right will align the text to the right.
  • .text-center will align the text to the center.
