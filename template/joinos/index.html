Overview

The Joinos Community has been built on the same framework as the Arro product. See Arro Styles Documentation for most core items to this product. All custom styles related to Jonios Community will be within this document along with a sample page created.

Theme

Custom Styling for the community

The joinos community CSS has been built so that only one class needs to be changed in order to completly change the pages look and feel.

Adding a custom class to the body element will style all custom elements within the page from one location. <body class="theme-red">

Available Themes

Class Theme
.theme-red Lets the page to a red theme
.theme-blue Lets the page to a blue theme
.theme-green Lets the page to a green theme
.theme-orange Lets the page to a orange theme
.theme-gray Lets the page to a gray theme

What is effected

.navbar he background colour of the navbar
.divider / .divider-thin The section dividers will automatically be upted to match the theme.
.btn-theme Button theme is added to each button that is related to the colour theme
