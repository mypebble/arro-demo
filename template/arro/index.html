Overview

This is the Template file for the updated SCSS for the Arro system, which include the migration from LESS to SASS.

Arro has been built on Bootstrap v3.3.7 SASS with a custom overlay

Layouts

How pages are structured

Grid Layout

Arro is based on the Bootstrap Grid system of using a maximum of 12 columns.

We use the class .container-fluid to allow the full page to be used by the user.

Tables Load Sample Table

Display blocks of information within a table

Tables are used to display data in a systematic order and will have a variety of styling applied to the table cell via a CSS Class. These classes may effect the following colour, size, text position, width.

Global Varients

Class Action
.table This class must be added to all tables to style them correctly
.table-hover This will apply a hover state to the table row
.table-bordered This will apply a border to the entire table including rows
.table-vam This class will align all fields to the middle of the cells (vertically)

Example

Table Head Table Head
Table Cell 

<table class="table table-bordered table-vam">
  <thead>
    <tr>
      <th>Table Head</th>
      <th>Table Head</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>
        Table Cell
      </td>
    </tr>
  </tbody>
</table>
